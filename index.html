Format Information

Code 128: Universal format supporting letters, numbers, and symbols. Great for general use.
Code 39: Alphanumeric format supporting uppercase letters and numbers. Widely used in industry.
EAN-13: International product barcode (13 digits). Used for retail products worldwide.
EAN-8: Compact version of EAN-13 (8 digits). Used for small retail products.
UPC-A: North American product barcode (12 digits). Standard for US/Canada retail.
ISBN-13: Book identification number (13 digits). Used for book publishing industry.
